# HTML — Mario Jump

# Aqui eu montei a estrutura do jogo.

Criei a área principal onde tudo acontece, coloquei as nuvens, o cano e o personagem.  
Também conectei o CSS e o JavaScript pra controlar o visual e a lógica.

Basicamente, o HTML é só o esqueleto do jogo.
Sem ele nada aparece na tela.

# CSS — Mario Jump

Aqui é onde eu fiz toda a parte visual do jogo.

Criei o cenário, posicionei os elementos e fiz as animações:
- pulo do personagem
- movimento do cano
- movimento das nuvens

Também ajustei tamanhos, posições e o fundo do jogo.

# JavaScript — Mario Jump

Aqui está toda a lógica do jogo.

Implementei:
- pulo quando aperta uma tecla
- movimento do obstáculo
- verificação de colisão
- parar o jogo quando perde
- imagem de game over

O JavaScript controla o que acontece durante o jogo.
Sem ele seria só uma animação sem interação.


O CSS é o que faz tudo se mexer e parecer um jogo de verdade.
